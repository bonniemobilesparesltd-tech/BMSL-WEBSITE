# BMSL-WEBSITE

Official website for **Bonnie Mobile Spares Ltd**.

## Overview
This is a professional, mobile-first business website designed to help customers contact the nearest branch via WhatsApp or phone and get directions instantly.

## Features
- **Branch Locator**: Detailed contact info and location links for all branches (Githurai, Ruiru, Nyeri, Kenol, Naivasha).
- **Direct Actions**: Click-to-call and WhatsApp integration.
- **Services Overview**: Spares, Repairs, and Accessories.
- **Responsive Design**: Optimized for mobile devices.

## Technologies
- React 19
- Tailwind CSS
- Lucide React Icons
